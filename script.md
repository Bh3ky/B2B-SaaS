## Setting up Backend

- run `uv init .`

- then `uv add`

- installing libraries and dependencies

`uv add fastapi uvicorn sqlalchemy python-dotenv pyjwt clerk svix`

## Running Backend

use `uv run start.py`

# Setting up Frontend

- `npm create@latest frontend`

- installing some packages 

`npm i @clerk/clerk-react react-router-dom`
